# MGA-Cotações

Comandos
--------
Criando o ambiente virtual:  $ python3 -m virtualenv myvenv 
Instalando os módulos:       $ (myvenv) pip install -r requirements.txt; 
Rodar os testes:             $ (myvenv) make
